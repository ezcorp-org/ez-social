**Page server (`src/routes/(app)/+page.server.ts`):**

This file does NOT exist yet (the current home page has no server load). Create it following the pattern from `src/routes/(app)/personas/+page.server.ts`.

**Load function:**
1. Get session via `event.locals.auth()` — if no user, return empty data
2. Get DB via `getDb(event.platform?.env?.DATABASE_URL ?? env.DATABASE_URL ?? "")` (use `$env/dynamic/private`)
3. Create queue service: `createQueueService(db)`
4. Create persona service: `createPersonaService(db)` (needed for persona list in quick-add dropdown)
5. Read `status` from `event.url.searchParams.get("status") ?? "all"`
6. Parallel fetch: `Promise.all([queueService.list(userId, status), queueService.getStatusCounts(userId), personaService.list(userId)])`
7. Return `{ posts, statusCounts, personas, activeStatus: status }`

**Form actions:**

`addPost` action:
1. Auth check (fail 401 if no session)
2. Parse formData: get `url` field
3. Validate with `addPostSchema.safeParse({ url })`
4. If validation fails, return `fail(400, { error: "Invalid URL" })`
5. Detect platform: `detectPlatform(url)`
6. Attempt scrape: `scrapeUrl(event.platform?.env?.SCRAPER_SERVICE_URL ?? env.SCRAPER_SERVICE_URL, url)` — handle with a single retry on failure
7. Auto-assign persona: `queueService.findPersonaByPlatform(userId, platform)` if platform detected, otherwise get default persona
8. Create post: `queueService.addPost(userId, { url, platform, personaId, postContent: scrapeResult?.content ?? null, postAuthor: scrapeResult?.author ?? null })`
9. If scrape returned null, return `{ success: true, needsContent: true, postId: post.id }` so UI shows manual paste fallback
10. Otherwise return `{ success: true }`

`updateContent` action (for manual content paste fallback):
1. Auth check
2. Parse formData: `postId`, `content`
3. `queueService.updateContent(userId, postId, content)`
4. Return `{ success: true }`

`assignPersona` action:
1. Auth check
2. Parse formData: `postId`, `personaId`
3. `queueService.updatePersona(userId, postId, personaId || null)`
4. Return `{ success: true }`
